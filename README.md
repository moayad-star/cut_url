# cut url

Free and open source tool
for link shortener

## Requirements

> python3.x 

> module requests

> module mechanicalsoup

> internet

## Install (linux,Termux)

```
git clone https://github.com/moayad-star/cut_url.git
```

```
cd cut_url
```

```
pip install -r requirements.txt
```

# Use


```python3 cut_url.py``` `<https://url>`
